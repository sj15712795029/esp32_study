# esp32_study
这是Wireless link ESP32教程开发板的仓库

|                | 2_uart_console        | ESP32 UART console工程                      |

| 3_lvgl         | 1_lvgl8_2_porting     | LVGL 8.2的移植                              |
|                | 2_lvgl8_0_porting     | LVGL 8.0的移植                              |
|                | 3_lvgl7_10_porting    | LVGL 7.10的移植                             |
